## ***What is this project about?***

*My first project in Hexlet. "Brain Gamesâ€” is a set of five console games. Each game asks questions that need to be answered correctly. The game is completed after three correct answers. Incorrect answers end the game and offer to go through it again. Games:*
+ ***Calculator:***  Arithmetic expressions that need to be calculated.
+ ***Progression:*** find the missing number in the arithmetic progression
+ ***Checking an even number***
+ ***Find the greatest common divisor***
+ ***Checking a prime number***
